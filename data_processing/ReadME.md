## Pipeline Execution Steps

1. **Update `pipeline.py`**  
   Open `pipeline.py` and modify:  
   - Crop name (e.g., `"WIWH"`)  
   - Crop type (`"winter"` or `"spring"`)  

2. **Run pipeline**  
   ```bash
   python pipeline.py
   ```

3. **Update `feature_engineering.py`**  
   Open `feature_engineering.py` and set crop name (e.g., `"WIWH"`)

4. **Run feature engineering**  
   ```bash
   python feature_engineering.py
   ```
